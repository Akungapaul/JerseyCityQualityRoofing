    **1. Create `src/hooks/use-scroll-past-fold.ts` (per CRO-03):**

    - Uses `useState(false)` and `useEffect` with `window.addEventListener("scroll", handler, { passive: true })`

    **2. Create `src/hooks/use-exit-intent.ts` (per CRO-05):**
    - `"use client"` directive at top
    - Module-level `let dismissed = false;` flag for session persistence (NOT inside component)
    - Export `useExitIntent(options?: { enabled?: boolean; delayMs?: number }): { isTriggered: boolean; dismiss: () => void }`
    - Default `enabled = true`, `delayMs = 5000`
    - Desktop detection: `document.addEventListener("mouseleave")` triggers when `e.clientY <= 0` AND `Date.now() - mountTime.current > delayMs`
    - Mobile detection: `window.addEventListener("scroll", { passive: true })` triggers when upward scroll delta > 300px AND dwell time > delayMs
    - `dismiss()` sets module-level `dismissed = true` and `setIsTriggered(false)`
    - Returns `{ isTriggered: isTriggered && !dismissed, dismiss }`
    - Both listeners skip if `!enabled || dismissed`
    - Uses `useRef` for `mountTime` (initialized to `Date.now()`) and `lastScrollY`
    - Cleanup removes both listeners on unmount
    - Internally checks `window.location.pathname` against allowed prefixes: `/services/` and `/service-areas/` -- if pathname does not start with either, hook never triggers (this keeps the marketing layout as a Server Component)
