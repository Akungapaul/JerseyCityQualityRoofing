## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, connector line width |
| sm | 8px | Compact element spacing, card internal gaps |
| md | 16px | Default element spacing, card padding, form field gaps |
| lg | 24px | Section inner padding (mobile), FAQ item padding |
| xl | 32px | Section heading margin-bottom, timeline step gaps |
| 2xl | 48px | Section vertical padding (mobile: py-12) |
| 3xl | 64px | Section vertical padding (desktop: py-16 to py-20) |

Exceptions:
- 44px minimum touch target on all interactive elements (buttons, accordion triggers, carousel controls) per WCAG 2.5.5 and existing button-variants.ts pattern
- 1280px max content width (established in SectionWrapper)
- px-4 / sm:px-6 / lg:px-8 horizontal page gutters (4/6/8 = 16/24/32px)

**Source:** Existing SectionWrapper pattern, button-variants.ts min-h-[44px].

## Typography

| Role | Size | Weight | Line Height | Font Family | CSS Class Pattern |
|------|------|--------|-------------|-------------|-------------------|
| Body | 18px (1.125rem) | 500 (medium) | 1.7 | Cormorant Garamond | `text-lg` (base body) |
| Label / Caption | 14px (0.875rem) | 500 (medium) | 1.5 | Cormorant Garamond | `text-sm` |
| Section Heading | 28px (1.75rem) | 700 (bold) | 1.2 | Cormorant | `text-[1.75rem] font-heading font-bold` |
| Page Title / Hero | 40px / 48px (2.5rem / 3rem) | 700 (bold) | 1.1 | Cormorant | `text-[2.5rem] lg:text-[3rem] font-heading font-bold` |

Additional typographic rules:
- Duration badges on timeline steps: 14px, weight 500, uppercase, letter-spacing 0.05em
- Material lifespan/price badges: 14px, weight 700
- Emergency hero phone number: 40px mobile / 56px desktop, weight 700, Cormorant
- All prose paragraphs: 18px at 1.7 line-height (matches body default)

**Source:** CLAUDE.md hard requirements (Cormorant Garamond medium, 18px min), globals.css (established values), existing component patterns (HeroSection, CTABanner, FaqAccordion).

## Interaction Patterns

### Scroll-Triggered Animations
- Every section's content wrapped in ScrollReveal (fade-in-up, 300ms duration)
- Respect prefers-reduced-motion (elements appear instantly, no animation)
- Animation triggers once per page load, not on re-scroll

### FAQ Accordion
- Single-open behavior (clicking one closes the other)
- First question open by default (defaultOpenIndex=0)
- Motion height animation (0 to auto) with 300ms easeOut
- Chevron rotates 180deg when open

### Material Card Expand/Collapse
- Pros/cons section hidden by default
- Toggle button reveals list with same Motion height animation pattern
- Multiple cards can be expanded simultaneously (independent state)

### TestimonialCarousel
- Embla carousel with autoplay (5000ms delay)
- Loop enabled, stops on mouse enter
- 1 slide visible on mobile, 3 on desktop (lg breakpoint)
- Previous/Next buttons with 44px touch targets

### Emergency Hero Phone Number
- Phone number is a `tel:` link (clickable on mobile)
- Visual weight: largest element in hero (40px mobile / 56px desktop)
- Emergency accent color (#d4782f) draws immediate attention

### Compact Form Auto-Selection
- CompactQuoteForm receives `defaultServiceType` prop matching current service name
- Service type dropdown is pre-selected, reducing friction

### Related Services Cards
- Hover: background darkens (secondary to secondary-lighter), arrow translates right 4px
- Entire card is a clickable link (next/link)
- Transition: 150ms (--duration-fast)

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Heading hierarchy | h1 = service name (in hero), h2 = section headings, h3 = item headings (FAQ questions, material names, process steps, warning sign titles) |
| Keyboard navigation | All interactive elements (accordion triggers, carousel controls, form inputs, CTA buttons, expand/collapse toggles) keyboard accessible |
| Focus visible | 2px accent ring with 2px dominant offset (established focus-ring pattern) |
| ARIA accordion | FAQ: role="region", aria-label, aria-expanded, aria-controls (existing pattern) |
| ARIA carousel | role="region", aria-roledescription="carousel", aria-label, slide groups with aria-roledescription="slide" (existing pattern) |
| Motion | prefers-reduced-motion respected globally (existing globals.css media query) |
| Touch targets | 44px minimum on all interactive elements |
| Color contrast | All text/background combinations meet WCAG AA. Text-primary (#f0ede6) on dominant (#2a2e22) = 11.2:1. Text-secondary (#b0ae9e) on dominant = 5.4:1. Accent (#c89640) on dominant = 5.1:1. Emergency accent (#d4782f) on dominant = 5.3:1. |
| Alt text | No images in this phase (content-driven pages). Lucide icons use aria-hidden="true". |
| Semantic HTML | main > section > headings. Ordered lists for process steps and emergency "What To Do". |
