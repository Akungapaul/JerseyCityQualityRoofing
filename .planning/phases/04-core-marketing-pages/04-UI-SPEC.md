## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (gap-1), inline micro-spacing |
| sm | 8px | Compact element gaps (gap-2), badge padding (py-1) |
| md | 16px | Default content padding (px-4), element spacing (gap-4) |
| lg | 24px | Container side padding at sm+ (px-6), card internal padding |
| xl | 32px | Container side padding at lg+ (px-8), layout gaps (gap-8) |
| 2xl | 48px | Section vertical padding mobile (py-12) |
| 3xl | 64px | Section vertical padding desktop (py-16), page-level breaks |
| 4xl | 80px | Section vertical padding lg+ (py-20), hero vertical padding |

Exceptions:
- Touch targets: 44px minimum (min-h-[44px] min-w-[44px]) per WCAG 2.1 Target Size -- applies to all buttons, links, interactive elements. Not a spacing token but enforced on all tap targets.
- Section container max-width: 1280px (max-w-[1280px]) -- matches SectionWrapper established in Phase 2.
- Header heights: 80px (full) / 56px (scrolled) -- matches header shrink-on-scroll from Phase 2.

**Source:** section-wrapper.tsx (py-12 sm:py-16 lg:py-20, px-4 sm:px-6 lg:px-8), button-variants.ts (min-h-[44px])

## Typography

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body | 18px (1.125rem) | 500 (medium) | 1.7 | Cormorant Garamond |
| Label / Caption | 18px (text-lg) | 500 (medium) | 1.5 | Cormorant Garamond |
| Section Heading (h2) | 28px (1.75rem) | 700 (bold) | 1.2 | Cormorant |
| Page Title / Hero (h1) | 40px (2.5rem) mobile, 48px (3rem) desktop | 700 (bold) | 1.1 | Cormorant |

**Constraints enforced:**
- Body text never goes below 18px (CLAUDE.md hard requirement)
- All button text uses text-lg (18px) per button-variants.ts
- Badge text uses text-lg (18px) per badge.tsx
- Heading hierarchy: one h1 per page (hero), h2 for section titles, h3 for sub-sections within sections. No h4+ needed for these marketing pages.

**Source:** globals.css body font-size (1.125rem), heading defaults (font-weight 700, line-height 1.2), cta-banner.tsx heading (1.75rem)

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #2a2e22 (Deep Olive) | Page backgrounds, section backgrounds (tone="dominant"), header bg |
| Dominant Lighter | #33382b | Badge strip background (slightly lighter than hero per D-06), hero overlay tinting |
| Secondary (30%) | #3e4435 (Olive Sage) | Alternating section backgrounds (tone="secondary"), card backgrounds, CTA banner bg, footer bg |
| Secondary Lighter | #4a5040 | Card borders, badge borders (border-secondary-lighter), subtle dividers |
| Accent (10%) | #c89640 (Warm Gold) | SEE RESERVED LIST BELOW |
| Accent Hover | #d8aa56 | Hover state for accent elements only |
| Text Primary | #f0ede6 (Warm White) | All body text, headings, primary content |
| Text Secondary | #b0ae9e (Muted Sage) | Supporting text, descriptions, timestamps, city card taglines |
| Destructive | #ef4444 (Red) | Form validation errors only |
| Success | #34d399 (Green) | Form submission success confirmation only |

**Accent reserved for (explicit list):**
1. Primary CTA buttons (bg-accent text-dominant) -- "Get Free Quote", "Request Free Inspection"
2. Phone number text color in header, CTA banner, and contact info
3. Star rating fill color on testimonial cards
4. Certification badge text color
5. Focus ring color (focus-visible:ring-accent)
6. Secondary button border and text color
7. Hover underlines on phone links
8. Link arrows on service grid cards and city cards

**Accent is NOT used for:** Section backgrounds, body text, heading text, borders on content sections, decorative elements, or icons that are not interactive.

**Source:** globals.css @theme block (Phase 2 approved palette: Deep Olive Sage), button-variants.ts, badge.tsx, cta-banner.tsx

## Interaction Contracts

### Testimonial Carousel (TestimonialCarousel)

| Property | Value |
|----------|-------|
| Library | Embla Carousel (embla-carousel-react 8.6.0 + embla-carousel-autoplay 8.6.0) |
| Slides visible (desktop) | 3 |
| Slides visible (mobile) | 1 |
| Loop | true |
| Autoplay delay | 5000ms |
| Stop on interaction | false (resumes after delay) |
| Stop on mouse enter | true (pauses while hovering) |
| Navigation | Prev/next buttons with ChevronLeft/ChevronRight Lucide icons |
| Accessibility | role="region", aria-roledescription="carousel", aria-label="Customer testimonials", each slide role="group" aria-roledescription="slide" |
| Slide content | 5-star rating (StarRating), quote text (~150 chars), customer name, city |

### FAQ Accordion (FaqAccordion)

| Property | Value |
|----------|-------|
| Library | Custom (no library -- native button + aria attributes) |
| Default open | First item (index 0) |
| Multiple open | No (single-open accordion) |
| Animation | Height transition via Motion (AnimatePresence) |
| ARIA pattern | button: aria-expanded, aria-controls; panel: role="region", aria-labelledby, hidden attribute |
| Heading level | h3 (wraps each trigger button) |
| Keyboard | Enter/Space to toggle, Tab to navigate between headers |
| Chevron | ChevronDown Lucide icon, rotates 180 degrees when expanded |

### Google Map Embed (GoogleMapEmbed)

| Property | Value |
|----------|-------|
| Loading | loading="lazy" on iframe (native lazy load) |
| Aspect ratio | 16:9 (aspect-video) |
| API key | NEXT_PUBLIC_GOOGLE_MAPS_API_KEY env var; falls back to query-based embed when empty |
| Accessibility | iframe title attribute describing map content |
| Fallback | If iframe fails, display text address with link to Google Maps |

### Hero Section (HeroSection)

| Property | Value |
|----------|-------|
| Layout | Split: left text (60%), right image/placeholder (40%) on desktop. Stacked on mobile (text above). |
| Background | Full-width bg-dominant |
| CTA buttons | Primary "Get Your Free Quote" (scrolls to #quote-form), Secondary phone call (click-to-call) |
| Hero placeholder | Gradient pattern with subtle roofing-themed SVG overlay. data-placeholder="hero-image" attribute for future replacement. |
| Animation | Fade-in-up on headline and CTAs via ScrollReveal (or direct Motion). No animation on initial viewport (above fold). |

### Badge Strip (BadgeStrip)

| Property | Value |
|----------|-------|
| Layout | Horizontal flex, center-aligned. Wraps on mobile (2-3 per row), single row on desktop. |
| Background | bg-dominant-lighter (#33382b) -- slightly lighter than hero per D-06 |
| Dividers | Subtle vertical dividers between items (1px border-secondary-lighter) |
| Items | Each: Lucide icon (20px) + text label |
| Animation | None (static, above fold on homepage) |

### Service Cards (ServicesGrid)

| Property | Value |
|----------|-------|
| Layout | 2 columns mobile, 4 columns desktop per D-03 |
| Grouping | "Residential Services" heading (h3) above residential row, "Commercial Services" heading (h3) above commercial row |
| Card content | Lucide icon (24px, text-accent), service name (font-bold), one-line description (text-secondary), ArrowRight icon link |
| Hover | Card background lightens to secondary-lighter, arrow icon shifts right 4px |
| Link target | /services/residential/[slug] or /services/commercial/[slug] |

### City Cards (CityCard / CityCardGrid)

| Property | Value |
|----------|-------|
| Tier 1 cards | Larger height, stronger visual weight. Show city name (h3), tagline, key stat, "View Services" link |
| Tier 2/3 cards | Standard height. Same content, more compact. |
| Hover | Card background lightens, subtle border-accent on hover |
| Link target | /service-areas/[slug] |
