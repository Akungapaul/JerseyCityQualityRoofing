  - H1: `<h1 className="font-heading font-bold text-[2.5rem] lg:text-[3rem] leading-[1.1] text-text-primary mt-4">{heroHeadline}</h1>`
  - Subtitle: `<p className="text-lg text-text-secondary mt-4 leading-relaxed">{heroSubtitle}</p>`
  - Population: `<p className="text-lg text-text-secondary mt-2">Serving {population.toLocaleString()} residents in {cityName}</p>`
  - CTAs container: `<div className="flex flex-wrap items-center gap-4 mt-8">`

**B. Create `src/components/sections/sibling-cities-nav.tsx`:**

A Server Component rendering horizontal navigation linking to the same service in all 12 Hudson County cities, with the current city visually distinct and non-clickable.

Props interface:
```typescript
interface SiblingCitiesNavProps {
  serviceName: string;
  serviceSlug: string;
  serviceCategory: 'residential' | 'commercial';
  currentCitySlug: string;
}
```

Structure:
- H2: `<h2 className="font-heading font-bold text-[1.75rem] text-text-primary mb-4">{serviceName} in Other Hudson County Cities</h2>`
- Nav element: `<nav aria-label="Same service in other cities" className="flex flex-wrap gap-3">`
- For each municipality from `getAllMunicipalities()`:
  - If `city.slug === currentCitySlug`: render `<span className="inline-flex items-center bg-secondary rounded-lg px-4 py-2 text-lg text-accent font-bold border border-accent cursor-default min-h-[44px]" aria-current="page">{city.name}</span>`
  - Else: render `<Link href={/services/${serviceCategory}/${serviceSlug}/${city.slug}} prefetch={false} className="inline-flex items-center bg-secondary rounded-lg px-4 py-2 text-lg text-text-primary border border-[#4a5040] hover:border-accent transition-colors duration-[150ms] min-h-[44px]">{city.name}</Link>`

Imports: `Link` from `next/link`, `getAllMunicipalities` from `@/data/municipalities`.

Note: Use `prefetch={false}` on all Link elements because there are 11 sibling links per page and 96 pages -- prefetching would create massive overhead.
